# Arrogant-Modpack

傲慢整合包（Arrogant-Craft）是一款以匠魂为核心的多模组科技冒险整合包，该整合包受到1.12.2贪婪整合包（Gready-Craft）影响，旨在为已经停更多年的贪婪制作高版本续作。

## 📦 内容介绍
- 整合多个热门模组
- 统一配置、提升兼容性
- 加强性能优化

## 🚀 使用方法
1. 将 Mod 放入 `.minecraft/mods` 文件夹
2. 打开 Minecraft 启动器
3. 选择正确版本启动即可

## 🔧 环境要求
- Minecraft 版本：1.21.1
- Java 17+
- Forge/Fabric（按你实际情况）
